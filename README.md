### 🍽️ [SwadSync](https://www.swadsync.in) — Offline-First Restaurant POS
> **Live in production · Real paying clients**

A dual-mode POS that runs as a fully offline Windows `.exe` AND a cloud PWA simultaneously — zero feature compromise in either mode.

- Bidirectional offline-to-cloud sync across **13 MongoDB collections** (last-write-wins conflict resolution)
- LAN-based real-time sync via **Socket.IO** — kitchen displays, cashier POS & waiter tablets in sync
- Full SaaS layer — Razorpay billing, RBAC across 5 roles, WhatsApp bill delivery
- Packaged MongoDB + Node.js into a single `.exe` via Electron — one-click install, no dependencies

`React` `TypeScript` `Node.js` `MongoDB` `Socket.IO` `Electron` `Redis` `Razorpay`

🔗 [Product](https://www.swadsync.in) · [Admin Dashboard](https://app.swadsync.in) · [POS Demo](https://pos.swadsync.in)
